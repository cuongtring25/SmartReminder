# 1. Overview:

Smart Reminder is an Android application designed to help users manage tasks efficiently by combining time-based and location-based reminders. 
Unlike traditional alarm apps, this app focuses on context-aware notifications and user behavior insights.
# 2. Core Features
- Time-based reminders
- Location-based reminders (GPS / Geofencing)
- Offline functionality
- Background processing
- Notification system
# 3. Advanced Features
- Gamification (points, levels, rewards)
- Daily Insight (task completion statistics)
- Streak system (track consecutive productive days)
- Smart Snooze (intelligent delay)
- Habit tracking elements
# 4. Streak System Logic
If a user completes at least 50% of their tasks in a day, the streak increases by 1. Otherwise, the streak resets to 0.
# 5. Technologies Used
- MVVM Architecture
- Room Database
- WorkManager / AlarmManager
- Google Maps & Location API
- Notification API
# 6. Project Goals
- Improve productivity
- Encourage habit building
- Provide smart and context-aware reminders
- Deliver a modern mobile-first experience
